About
Who are we?

  We are an Indie game and software development team founded by Middle and High school students from Turkey.

 We started our career with a weekly game jam and made that 'Horrible' game on 3 September 2020, when we were 14. Since this day we participate in a lot of game jams and competitions, in this path we earned a lot of friends.
Which tools are we using?
 We generally use ' Unity ' as our Game Engine, ' Blender ' for 3d modelling and UV editing, 'Krita' or 'Photoshop' for 2D art and illustrations, ' Aseprite ' for pixel arts and ' Audacity ' for sound design.

 For the IDE's things are a little complicated because everyone has an IDE comfortable with them. So It's your choice, Try them!
